The module contains a class MyList that:
    - Inherits from the built-in list class.
    - Implements a method print_sorted() that prints a list in sorted ascending order.

>>> my_list = MyList([4, 1, 6, 3])
>>> my_list.print_sorted()
[1, 3, 4, 6]

>>> my_list.append(0)
>>> my_list.print_sorted()
[0, 1, 3, 4, 6]

>>> my_list.append(5)
>>> my_list.print_sorted()
[0, 1, 3, 4, 5, 6]
